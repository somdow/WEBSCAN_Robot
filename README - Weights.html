Scoring Formula
Each module result gets a multiplier: Ok = 1.0, Warning = 0.5, Bad = 0.0, Info = excluded.

Overall Score = sum(weight × multiplier) / sum(weight) × 100

Example: If SSL Certificate (20) is Bad and Title Tag (15) is Ok, the score loses 20 points from the pool but keeps 15. Modules with weight 0 are displayed in results but do not affect the score.
