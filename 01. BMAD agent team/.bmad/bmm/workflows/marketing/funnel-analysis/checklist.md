# Funnel Analysis Validation Checklist

## Funnel Definition
- [ ] Funnel type identified
- [ ] All stages defined
- [ ] Entry/exit criteria clear
- [ ] Analysis period specified
- [ ] Segments to analyze identified

## Current State
- [ ] Volume per stage documented
- [ ] Conversion rates calculated
- [ ] Drop-off percentages calculated
- [ ] Time in stage analyzed
- [ ] Overall conversion documented

## Drop-Off Analysis
- [ ] Biggest drop-offs identified
- [ ] Root causes hypothesized
- [ ] Contributing factors listed
- [ ] Drop-offs prioritized by impact

## Segmentation
- [ ] By acquisition source
- [ ] By device type
- [ ] By user segment (if applicable)
- [ ] Insights extracted per segment

## Benchmarking
- [ ] Industry benchmarks researched
- [ ] Internal trends analyzed
- [ ] Gaps quantified
- [ ] Uplift potential calculated

## Recommendations
- [ ] Prioritized by impact/effort
- [ ] Hypothesis stated for each
- [ ] Actions are specific and actionable
- [ ] Expected impact quantified

## Report Quality
- [ ] Funnel visualized clearly
- [ ] Testing roadmap realistic
- [ ] Impact projections reasonable
- [ ] Next steps clear
