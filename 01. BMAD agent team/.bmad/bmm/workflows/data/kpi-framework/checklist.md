# KPI Framework Validation Checklist

## Business Context
- [ ] Business objectives documented
- [ ] Success criteria defined
- [ ] Stakeholders identified
- [ ] Decision contexts mapped
- [ ] Business model understood

## North Star Metric
- [ ] Single metric identified
- [ ] Reflects customer value
- [ ] Leading indicator confirmed
- [ ] Actionable by teams
- [ ] Formula defined

## Metrics Hierarchy
- [ ] 3-5 strategic KPIs defined
- [ ] Tactical metrics per KPI
- [ ] Clear ownership assigned
- [ ] Hierarchy visualized
- [ ] No vanity metrics

## Targets and Benchmarks
- [ ] Baselines established
- [ ] Targets are SMART
- [ ] Industry benchmarks included
- [ ] Stretch goals defined
- [ ] Thresholds documented

## Measurement Strategy
- [ ] Data sources identified
- [ ] Formulas documented
- [ ] Tracking frequency set
- [ ] Segmentation defined
- [ ] Alerting rules established

## Report Quality
- [ ] Framework is actionable
- [ ] Governance model clear
- [ ] Implementation roadmap realistic
- [ ] Review cadence established
