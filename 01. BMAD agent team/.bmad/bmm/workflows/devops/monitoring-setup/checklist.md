# Monitoring Setup Validation Checklist

## Metrics
- [ ] Infrastructure metrics defined
- [ ] Application metrics (RED) defined
- [ ] Business metrics identified
- [ ] Collection method specified
- [ ] Dashboards designed

## Logging
- [ ] Log format defined (structured)
- [ ] Log levels documented
- [ ] Correlation IDs included
- [ ] No PII in logs
- [ ] Aggregation solution selected
- [ ] Retention policy defined

## Tracing
- [ ] Trace propagation defined
- [ ] Services instrumented
- [ ] Sampling strategy selected
- [ ] Cross-service correlation works

## Alerting
- [ ] Severity levels defined
- [ ] Alert channels configured
- [ ] SLOs defined
- [ ] Actionable alerts only
- [ ] Runbooks linked
- [ ] On-call rotation addressed

## Tools
- [ ] Tool stack selected
- [ ] Configuration provided
- [ ] Integration points defined
- [ ] Access control considered

## Documentation
- [ ] Architecture documented
- [ ] Implementation plan provided
- [ ] Runbooks created
