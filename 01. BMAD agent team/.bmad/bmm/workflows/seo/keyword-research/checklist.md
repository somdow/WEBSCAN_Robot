# Keyword Research Validation Checklist

## Research Completeness

### Scope Definition
- [ ] Target niche/industry clearly defined
- [ ] Target audience documented
- [ ] Geographic scope specified
- [ ] Business goals understood

### Keyword Discovery
- [ ] Seed keywords identified
- [ ] Long-tail variations explored
- [ ] Question keywords included
- [ ] Comparison keywords explored
- [ ] Problem-based keywords considered
- [ ] Minimum 30 keywords researched

### Intent Classification
- [ ] All keywords classified by intent
- [ ] Informational keywords identified
- [ ] Commercial keywords identified
- [ ] Transactional keywords identified
- [ ] Intent matches recommended content type

### Competition Analysis
- [ ] Top results analyzed for priority keywords
- [ ] Difficulty scores assigned
- [ ] SERP features documented
- [ ] Quick wins identified
- [ ] Unrealistic targets flagged

### Prioritization
- [ ] Clear prioritization criteria applied
- [ ] Top 10 priorities highlighted
- [ ] Quick wins separated from long-term bets
- [ ] Each priority has recommended content type

### Content Clusters
- [ ] At least one content cluster defined
- [ ] Pillar content identified per cluster
- [ ] Supporting content mapped
- [ ] Internal linking strategy suggested

## Quality Checks

- [ ] No keyword stuffing recommendations
- [ ] Intent matches content recommendations
- [ ] Difficulty assessments are realistic
- [ ] Recommendations are actionable
- [ ] Executive summary captures key insights
- [ ] Action plan has clear next steps
