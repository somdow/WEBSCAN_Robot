# Auth Review Validation Checklist

## Password Security
- [ ] Password policy documented
- [ ] Hashing algorithm verified
- [ ] Salt usage confirmed
- [ ] Password reset flow reviewed
- [ ] Account enumeration checked

## Session Management
- [ ] Cookie flags reviewed
- [ ] Session timeout configured
- [ ] Logout invalidation verified
- [ ] Session fixation addressed
- [ ] Concurrent sessions handled

## Token Security
- [ ] Algorithm strength verified
- [ ] Expiration times appropriate
- [ ] Refresh token rotation
- [ ] Revocation strategy exists

## Authorization
- [ ] Access control model documented
- [ ] Every endpoint protected
- [ ] IDOR vulnerabilities checked
- [ ] Privilege escalation tested
- [ ] Default deny confirmed

## Report Quality
- [ ] All findings rated
- [ ] Recommendations actionable
- [ ] Quick wins identified
- [ ] Architecture documented
